# Deadbolt Overview

Deadbolt is a privacy-preserving technology concept for proximity-based public-health and safety infrastructure.

The public framing is intentionally high level: decentralised proximity logging, privacy preservation, and interoperability across communities or jurisdictions.

## Public principles

- Preserve privacy by avoiding unnecessary identity disclosure.
- Support interoperability across organisations or borders.
- Keep implementation mechanisms and unpublished invention detail out of public repositories.
- Use this repository as a clean public overview, not a legacy website dump.
